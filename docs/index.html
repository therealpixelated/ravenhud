Security & Transparency

Every release is automatically verified for your safety.

Code Signing

Every .exe is signed with Microsoft Azure Trusted Signing. Right-click the installer, select Properties, then Digital Signatures to verify.

VirusTotal Scan

Each release is scanned against 70+ antivirus engines. View the scan link in release notes or upload to virustotal.com yourself.

Open Source

Source code is available for review. Check our OSSF Scorecard for security best practices.

How to Verify Your Download

  1. Check signature: Right-click .exe → Properties → Digital Signatures → Should show "therealcloudvikinggmail.onmicrosoft.com"
  2. VirusTotal: Each release includes a link to the scan results in the release notes
  3. Hash check: Compare the SHA256 hash shown in release notes
